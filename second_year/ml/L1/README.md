# Machine Learning Labs – Setup Guide

## 📚 Introduction
This guide provides the necessary steps to set up your Python environment, install Jupyter Notebook, and import the **L1.ipynb** notebook for your first lab session.

---

## ✅ Requirements
Before you begin, ensure you have:
- A computer with **Windows, macOS, or Linux**.
- A stable internet connection.
- Basic knowledge of Python.
- (Optional) Visual Studio Code installed for an alternative environment.

---

## 🛠️ Step 1: Install Python Environment
We recommend installing the **latest stable version of Python**.  
Follow the instructions based on your operating system:

## 📝 Step 2: Install Jupyter Notebook
Once Python is installed, install **Jupyter Notebook** using `pip`:

```sh
pip install jupyter
```

---

## 🚀 Step 3: Open and Run the Lab Notebook (L1.ipynb)
You can choose one of the following methods to open and run the lab notebook:

### **Option 1: Using Jupyter Notebook**
1. Open a terminal and start Jupyter Notebook by running:
   ```sh
   jupyter notebook
   ```
2. This will launch a **web interface** in your default browser.
3. Upload the **L1.ipynb** file by navigating to its location and clicking "Upload".
4. Open the notebook and start running the cells.

### **Option 2: Using Visual Studio Code**
1. Open **VS Code** and install the **Jupyter extension** if not already installed.
2. Open the `L1.ipynb` file within VS Code.
3. Try running a cell; follow the prompts to install any missing dependencies.
4. Once the dependencies are installed, proceed with running the notebook.

---

## 🎯 Next Steps
Once you've successfully set up Jupyter Notebook and opened **L1.ipynb**, you can begin working on the **first lab**. Make sure to follow the instructions inside the notebook.

Happy coding! 🚀
